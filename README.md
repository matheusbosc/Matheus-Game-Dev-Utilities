# Matheus' Game Development Utilities
By Matheus  -  [matheusbosc.com](https://matheusbosc.com)

This package helps speed up your game development journey by providing you with many systems and pre-made scenes to speed up setting up your projects.

## The Main Menu System
Implementing a main menu system can sometimes be time-consuming. This package implements a pre-made main menu scene. It integrates with the rest of the package to create a seamless experience when heading into the game.

This system has a few pages. The main, title page, a play page featuring different save slots for the save system, a settings page to manage volume, graphics, and more, and a credits page so your team can be recognized.


## Log System
The log system creates a simple way to log to a custom file. This includes 4 types of logs: log, warning, error, and debug. You can access the log system throughout your project to help you and your users debug your game.


## Save System
The save system provides an easy way to save your game state with a simple object. It saves an encrypted JSON file (or decrypted if you chose) containing all of the save information to your app data.


## Scene Manager & Loading Screen
Load your scenes and sync your data with ease using the Scene Manager and display progress through a highly customizable loading screen. Load scenes asynchronously and initialize your scenes while tracking load progress.


## Example
To view the example without much setup, clone this repository and just click play.
